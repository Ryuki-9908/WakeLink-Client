# WakeLink Client

WakeLink Client は、Wake-on-LAN 機能と SSH クライアント機能を統合した、クロスプラットフォーム対応の軽量ツールです。  
自宅や社内のサーバー／PCを遠隔から起動し、そのままSSH接続するまでの一連の流れを、シンプルなUIで提供します。

---

- その他詳細機能の実装

## 特徴

- **Wake-on-LAN 対応**  
  任意のMACアドレス宛にマジックパケットを送信して、リモート端末を起動できます。

- **SSH接続の簡略化**（実装予定）  
  起動後、指定されたIPとポートに自動的にSSH接続します。

- **クロスプラットフォーム**  
  Windows / Linux 両対応（動作確認済）。OSを問わず同一UIで操作可能です。

- **GUI操作中心**  
  ターミナル操作に不慣れなユーザーでも直感的に使える設計。

---

## 開発背景

日常業務におけるリモート操作の効率化を目的に、Wake-on-LAN→SSH接続の一連の流れを自動化できるクライアントとして開発を開始しました。  
「直感的かつ安全に使えるツール」を目指し、GUI・設定項目の柔軟性・誤操作の防止設計を重視しています。
